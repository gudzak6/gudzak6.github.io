"RunTheWorld" (Working title)

Running Copenhagen showed me the amount of demand there is for runners that want to travel and race! My goal is to create an experience for runners that will provide them with the ability to travel abroad and run a race with a group of people. Users will be able to follow a coach-created program, train with like minded individuals, and eventually run for a PR in an awesome race.

RunBFF finder

A problem I routinely run into which is finding a suitable training partner. With the NYC running scene being so competitive and large, I want to build a platform that connects runners for workouts. Picture what TripBFF is but solely for runners looking for people to run with in different cities or areas!
